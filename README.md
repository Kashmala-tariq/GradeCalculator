
# GradeCalculator
A Java Swing GUI application to calculate CGPA and Grades.

## Description
This is a desktop-based Grade & CGPA Calculator built using Java and Swing. 
It takes subject names, credit hours, and marks as input, then calculates weighted CGPA on a 4.0 scale with detailed grading from A+ to F.

## Features
- **Multiple Subjects**: Add unlimited subjects with credit hours
- **Accurate CGPA**: Calculates weighted CGPA like university system
- **Grade Report**: Shows A+, A, B+, B, C+, C, D+, D, E, F for each subject
- **Clean GUI**: Well-defined interface with table format output
- **Easy to Use**: Add, Calculate, and Clear buttons

## How to Run
```bash
javac GradeCalculator.java
java GradeCalculator
